== Installation ==
<h3>Please complete the following steps to get the TrafficTrain plugin working on your site:</h3>
1. Upload the entire TrafficTrain' plugin directory to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click on the TrafficTrain menu link that appears along the left menu bar and
1. Follow the onscreen instructions on how to get a TrafficTrain account and code snippet
== Frequently Asked Questions ==
<h3>Is a TrafficTrain.com account required?</h3>
Yes. In order to get the campaign code to use with the plugin, a user must first setup a free TrafficTrain.com account. Signup for an account today and the first $10 in customer leads is on us.
<h3>Can I cancel my account?</h3>
Yes and no. If you have no funds in your account and no running campaigns we can cancel/close your account. However if there are funds in the wallet, the account remains open until the funds are used.
